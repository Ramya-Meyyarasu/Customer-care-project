# 📊 Call Center Data Analysis (Excel Project)

## 🧰 Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Dashboards
- Slicers

 ## 📁 Dataset
The dataset includes call center details such as:
- Number of Calls
- Customer Satisfaction Rating
- Representative Name
- Call Amount/Revenue

## 🎯 Project Objective
To analyze call center performance and identify top-performing representatives, customer satisfaction levels, and revenue trends.

## 📊 Key Metrics Analyzed
- Total Number of Calls Handled
- Average Customer Satisfaction Rating
- Revenue (Amount) Generated
- Representative-wise Call Handling Performance

## 🔍 Key Insights
- Identified the representative who handled the highest number of calls
- Compared satisfaction ratings to evaluate service quality
- Analyzed revenue contribution by each representative
- Observed relationship between number of calls and customer satisfaction

## 📈 Dashboard Features
- Interactive dashboard using slicers
- Filter by representative to identify who handled the most calls
- Visual comparison of satisfaction ratings across representatives
- Revenue analysis based on call performance
  
## ✅ Conclusion
This project demonstrates how Excel can be used to analyze call center data, track performance, and generate insights for improving customer service and business outcomes.

# 📊 Call Center Data Analysis (Excel Project)

## 🧰 Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Dashboards
- Data Cleaning

 ## 📁 Dataset
The dataset contains call center records including:
- Call Date & Time
- Call Duration
- Customer Satisfaction Score

## 🎯 Project Objective
To analyze call center performance and identify key insights to improve customer service.

## 📊 Key Analysis Performed
- Total number of calls handled
- Average call duration
- Call volume trends (daily/monthly)
- Customer satisfaction trends

## 🔍 Key Insights
- Found peak call hours during the day
- Observed average handling time trends
- Highlighted areas with low customer satisfaction

## 📈 Dashboard Features
- Interactive Excel dashboard
- KPI metrics (Total Calls, Amount ,Rating)
- Visual charts for trends and comparisons

## ✅ Conclusion
This project helps in understanding call center operations and provides actionable insights to improve efficiency and customer experience.
